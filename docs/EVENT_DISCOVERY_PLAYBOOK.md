## Database Schema Notes

When adding events, use these categories:
- `fair` - Art fairs (Zona Maco, Material, ACME, BADA, Territorio, Laguna)
- `opening` - Gallery opening receptions
- `exhibition` - Ongoing exhibitions
- `party` - Nightlife events
- `experience` - Listening rooms, salons, brunches
- `talk` - Panels, conversations
- `performance` - Live art

Sources used:
- `whitewaller` - Whitewaller guide
- `manual` - Manually added
- `excelsior`, `cultivarte`, etc. - Spanish media
- `ocula`, `myartguides` - Aggregators

## Lessons Learned

1. **Spanish sources are essential** - Excelsior, CultivArte have events English sources miss
2. **Design/art blur is real** - Check design publications, not just art
3. **Galleries don't always announce early** - Some exhibitions only appear on Instagram
4. **Party info comes last** - Often announced 1-2 weeks before
5. **WhatsApp is crucial** - Local knowledge shared in groups
6. **Check individual gallery sites** - Aggregators miss some shows
7. **Hotels have programming** - Design Hotels, Habita group especially
8. **Burning Man camps are a thing** - Mayan Warrior, MAXA, La Calaca, Sacbe
