## Event Intake Process

### For Each Event Found:
1. **Capture**: Screenshot or save URL immediately
2. **Extract**: Name, date, time, venue, address, price, ticket URL
3. **Categorize**: Party, exhibition, gallery, talk, workshop, fair, experience
4. **Verify**: Cross-reference with venue/artist social media
5. **Import**: Add to events.json and run import script

### Quality Criteria
- Must have: Name, date, venue
- Should have: Time, price, ticket URL
- Nice to have: Description, image, neighborhood
