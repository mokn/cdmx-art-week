## Notes for Future Years

1. **Whitewaller guide** - Published ~2 weeks before Art Week, most comprehensive single source
2. **Spanish media** - Often has events 1-2 weeks before English sources
3. **Gallery websites** - Check individually, aggregators miss some shows
4. **Instagram** - Essential for parties, often only source for last-minute events
5. **WhatsApp** - Get added to local groups for insider info
6. **PDF guides** - Save locally, often disappear after event
